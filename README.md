Cereal
===========

Stream & download your favorite shows


## Install
```
$ npm install
```

## Start
```
$ npm start
```

### Create OS specific distributable
```
$ npm i -g electron-builder
$ npm i -g electron-packager
$ npm install --global --production windows-build-tools
$ npm run dist
```
